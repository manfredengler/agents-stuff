# Catálogo de creaciones

Espacio para **archivar** y **compartir** creaciones relacionadas con agentes.

## Estructura

- [`skills/`](./skills/)
- [`plugins/`](./plugins/)
- [`conectores/`](./conectores/)
- [`prompts/`](./prompts/)
- [`system-prompts/`](./system-prompts/)
- [`hooks/`](./hooks/)
- [`mcp/`](./mcp/)
- [`otros/`](./otros/)

## Cómo usar este catálogo

1. Crear o copiar tu recurso en la carpeta correspondiente.
2. Añadir una entrada en el `README.md` de esa categoría.
3. Incluir nombre, descripción breve y enlace al recurso.

## Plantilla de entrada

| Nombre | Descripción | Autor | Enlace |
|---|---|---|---|
| Ejemplo | Qué hace y para qué sirve | @usuario | `./ruta/al/recurso` |
