# Prompts

Colección de **plantillas de prompt** para el usuario final.

## ¿Qué es un prompt aquí?

Un prompt es una plantilla de instrucción o consulta lista para usar que el usuario puede enviar al agente. Puede incluir variables, instrucciones de contexto, o ejemplos de salida esperada. No confundir con system prompts (instrucciones de sistema para el agente).

## Cuándo usar esta categoría

- El recurso es una plantilla de texto que el usuario escribe o pega en el chat.
- Sirve para guiar al agente en una tarea concreta y repetible.
- Puede incluir marcadores de posición (`{{variable}}`) para personalizar.

## Buenas prácticas

- Incluye una descripción del caso de uso y el resultado esperado.
- Especifica si el prompt requiere contexto adicional (archivos abiertos, selección, etc.).
- Usa `{{variable}}` para los campos personalizables y documenta qué representa cada uno.
- Proporciona al menos un ejemplo de entrada/salida.
