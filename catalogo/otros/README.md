# Otros

Recursos que no encajan en ninguna de las categorías principales.

## Cuándo usar esta categoría

- El recurso es claramente útil para la comunidad pero no corresponde a ninguna de las categorías existentes.
- Es experimental o difícil de clasificar de forma inequívoca.
- Combina características de múltiples categorías de forma inseparable.

## Buenas prácticas

- Explica brevemente en el `README.md` del recurso por qué no cabe en otra categoría.
- Si crees que el recurso representa una nueva categoría emergente, abre una issue para proponer añadirla al catálogo.
- No uses esta categoría como cajón de sastre habitual; úsala solo si ninguna otra categoría aplica.
