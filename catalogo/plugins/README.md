# Plugins

Colección de **extensiones y plugins** para VS Code y GitHub Copilot.

## ¿Qué es un plugin?

Un plugin es una extensión que se instala en el entorno de desarrollo (VS Code, GitHub Copilot Chat, etc.) para añadir nuevas funcionalidades: comandos, vistas, proveedores de contexto, etc.

## Cuándo usar esta categoría

- El recurso se instala como extensión de VS Code o Copilot.
- Añade comandos, paneles, o integraciones al entorno de desarrollo.
- No es un skill invocable por el agente en tiempo de ejecución.

## Buenas prácticas

- Especifica la versión mínima de VS Code o Copilot requerida.
- Incluye instrucciones de instalación en el `README.md` del plugin.
- Indica si el plugin requiere configuración adicional (tokens, settings).
